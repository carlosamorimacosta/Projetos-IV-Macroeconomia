# 📊 Impactos Macroeconômicos das Tarifas dos EUA sobre o Brasil

Este repositório reúne o projeto desenvolvido por alunos do 3º ano de Economia da **FGV EESP**, com foco na análise dos impactos do aumento de tarifas dos Estados Unidos sobre produtos brasileiros, conhecido como **“Tarifaço EUA–Brasil”**.

A proposta busca compreender os efeitos dessa política sobre a economia brasileira, combinando **análise de dados**, **teoria econômica** e **evidência empírica**.

## 🎯 Objetivos

- Avaliar impactos sobre exportações brasileiras  
- Analisar efeitos na balança comercial, câmbio e inflação  
- Identificar setores mais afetados  
- Propor recomendações econômicas e estratégicas  

## 🛠️ Metodologia

- **Análise de Dados:** Uso de bases de comércio exterior e indicadores macroeconômicos  
- **Estatística:** Estatísticas descritivas e análise de séries temporais  
- **Revisão de Literatura:** Estudos sobre tarifas, protecionismo e comércio internacional  
- **Fontes de Dados:** COMEX Stat, IBGE, Banco Central, entre outras  

## 📁 Conteúdo

O repositório está organizado com os principais materiais do projeto:

- **Dados:** Bases brutas e tratadas utilizadas nas análises  
- **Notebooks / Scripts:** Códigos para limpeza, análise e visualização dos dados  
- **Relatórios:** Documentos com análises e interpretações econômicas  
- **Apresentações:** Slides com síntese dos resultados  
- **Outputs:** Gráficos, tabelas e resultados gerados
